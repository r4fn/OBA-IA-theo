# Quiz OBA – Nível 2

Quiz interativo de estudo para a **Olimpíada Brasileira de Astronomia e Astronáutica (OBA)** — Nível 2, destinado a alunos do 4º e 5º ano do ensino fundamental.

20 questões geradas com base nos padrões das provas de 2020 a 2024, cobrindo **Astronomia** e **Astronáutica**.

## Funcionalidades

- Resposta com feedback imediato (certo/errado)
- Explicação didática a cada questão
- Barra de progresso e contador de acertos
- Tela final com pontuação e estrelas
- Tema espacial com estrelas animadas
- Responsivo para celular e tablet

## Rodar localmente

```bash
npm install
npm run dev
```

## Deploy no GitHub Pages

O deploy é automático via GitHub Actions a cada push na branch `main`.

**Para ativar o GitHub Pages no repositório:**

1. Vá em **Settings → Pages**
2. Em *Source*, selecione **GitHub Actions**
3. Faça um push na branch `main`
4. Aguarde o workflow terminar — o link do site aparecerá em Settings → Pages
